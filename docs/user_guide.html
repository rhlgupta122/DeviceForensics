Installation & Setup

Prerequisites

  • Python 3.8 or higher
  • Windows 10/11 operating system
  • Administrator privileges (recommended for full access)
  • Minimum 4GB RAM (8GB recommended)
  • At least 2GB free disk space

Installation Steps

# Clone or download the repository git clone https://github.com/your-repo/DeviceForensics.git cd DeviceForensics # Create virtual environment python -m venv .venv # Activate virtual environment # Windows: .venv\Scripts\activate # Linux/Mac: source .venv/bin/activate # Install dependencies pip install -r requirements.txt
Important: Some features require administrator privileges to access system files and registry keys. Run the application as administrator for best results.

Optional Dependencies

# For EVTX file analysis (optional) pip install evtx # For enhanced memory analysis (optional) pip install volatility3

Command Line Interface (CLI)

The command line interface provides powerful automation capabilities and follows KAPE-style syntax for familiarity with forensic professionals.

Basic Usage

# Launch GUI python main.py --gui # Extract all artifacts python main.py --extract-all --output-dir ./forensic_output # Extract specific artifacts python main.py --registry --filesystem --memory --output-dir ./output # Extract with CSV export python main.py --extract-all --output-dir ./output --export-csv # Verbose logging python main.py --extract-all --output-dir ./output --verbose

Command Line Options

Option Description Example --gui Launch graphical user interface python main.py --gui --extract-all Extract all available artifacts python main.py --extract-all --registry Extract registry artifacts python main.py --registry --filesystem Extract file system artifacts python main.py --filesystem --memory Extract memory artifacts python main.py --memory --network Extract network artifacts python main.py --network --evtx Extract EVTX artifacts python main.py --evtx --output-dir Specify output directory --output-dir ./forensic_output --export-csv Export artifacts to CSV format --export-csv --hash-algorithm Specify hash algorithm (md5, sha1, sha256, sha512) --hash-algorithm sha256 --log-level Set logging level (DEBUG, INFO, WARNING, ERROR) --log-level INFO --verbose Enable verbose output --verbose

Troubleshooting

Common Issues

Permission Denied Errors

Problem: "Permission denied" or "Access denied" errors during extraction

Solution: Run the application as administrator for full system access

EVTX Library Not Found

Problem: "EVTX library not available" error

Solution: Install the EVTX library: pip install evtx

Application Hanging

Problem: Application appears to hang during extraction

Solution: The application has built-in limits to prevent hanging. Check logs for "maximum file limit" messages

Missing Registry Keys

Problem: "The system cannot find the file specified" for registry keys

Solution: This is normal for keys that don't exist on the system. The application handles this gracefully

Performance Optimization

  • Reduce Max Files: Lower the max files limit if extraction is slow
  • Selective Extraction: Extract only needed artifact types
  • SSD Storage: Use SSD storage for faster file access
  • Sufficient RAM: Ensure adequate RAM for large extractions
